# Catgirl Dock App (macOS)

Мини‑приложение для macOS: в Dock появляется анимированная ASCII/ANSI‑иконка кошкодевочки.

## Что делает

- Показывает кошкодевочку в доке из символов (моноширинный рендер в `NSImage`).
- Постоянно «дышит» в idle-анимации.
- Реагирует на нажатия клавиш: на каждое нажатие запускается короткая анимация, где кошкодевочка машет лапками.

## Запуск

```bash
swift run
```

После старта приложение появится в Dock. Для глобального отслеживания клавиатуры macOS может запросить разрешения в `Privacy & Security`.

## Технически

- `AppKit` + `NSApplication` для фонового desktop-приложения.
- `NSEvent` local/global monitor для реакции на клавиатуру.
- `Timer` для кадровой анимации.
- Отрисовка иконки в `NSImage` с моноширинным шрифтом.
