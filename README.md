# app (macOS)

Мини‑приложение для macOS: в Dock появляется анимированная ASCII/ANSI‑иконка кошкодевочки.

## Что делает

- Показывает кошкодевочку в доке из символов (моноширинный рендер в `NSImage`).
- Постоянно «дышит» в idle-анимации.
- Реагирует на нажатия клавиш: на каждое нажатие запускается короткая анимация, где кошкодевочка машет лапками.

## Быстрый запуск через SwiftPM

```bash
swift run app
```

## Сборка готового `.app`

Чтобы получить обычный macOS bundle, запусти скрипт:

```bash
./scripts/build-app.sh
```

После этого появится готовое приложение:

- `dist/app.app`

Запуск на macOS:

```bash
open dist/app.app
```

> Если macOS блокирует запуск из‑за подписи, можно подписать своим Developer ID или использовать ad-hoc подпись. Скрипт уже пытается поставить ad-hoc подпись автоматически, если `codesign` доступен.

## Скачать готовый app

Готовый `app.app` собирается в GitHub Actions на macOS.

1. Открой вкладку **Actions**.
2. Запусти workflow **Build macOS app bundle** (кнопка **Run workflow**).
3. После выполнения можно скачать архив двумя способами:
   - из артефакта `app-macos` в Actions;
   - прямо из репозитория: `prebuilt/app-macos.zip`.
4. Распакуй архив и запусти `app.app`.

> На `Run workflow` CI также коммитит свежий `prebuilt/app-macos.zip` обратно в репозиторий.

## Технически

- `AppKit` + `NSApplication` для фонового desktop-приложения.
- `NSEvent` local/global monitor для реакции на клавиатуру.
- `Timer` для кадровой анимации.
- Отрисовка иконки в `NSImage` с моноширинным шрифтом.
